# 🔐 VulnVision Lite - Vulnerability Scanner

## 📌 Description
VulnVision Lite is a basic cybersecurity tool that performs vulnerability scanning on a target system.

## 🚀 Features
- Port scanning (detect open ports)
- Security header analysis
- Report generation
- Logging system

## 🛠 Technologies Used
- Python
- Socket Programming
- Requests Library

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

3. Enter target (example.com)

## 📊 Output
- Displays open ports
- Shows missing security headers
- Generates a report file

## 🎯 Objective
To simulate real-world vulnerability scanning similar to tools like OpenVAS.
